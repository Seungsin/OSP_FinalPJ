Welcome our Analysis page!

How to use it

When you enter url address or upload a file with url input, we analyze them and inform them of similarity, number of duplicate words, etc.
Press the start button to go to the second page. On the second page, you can choose between search and file to enter url.
Press each button to go to the appropriate page, which can be passed by entering the scroll of the mouse.
Of course, you can always come to the main screen through the back to main button.
